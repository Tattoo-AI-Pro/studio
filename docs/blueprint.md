# **App Name**: Tattoo AI-Book Factory

## Core Features:

- AI-Book Creation: Enable the user to create AI-Books by defining the name, description, target audience, and tags.
- Module Creation: Allow users to create modules within each AI-Book, defining a name, sub-description and an estimated number of images to include.
- Image Upload and AI Analysis: Automatically classify images by theme and style and also generate suggested names, copy, SEO tags and instagram captions, while allowing user adjustments if needed.
- Visual Editor: Provide a drag-and-drop style editor (like Notion/Canva) to reorder modules, adjust pages, and edit the AI-Book's copy.
- AI-Book Compilation: Compile the AI-Book into a PDF, a responsive web version, a mini-site for sales, promotional files, marketing copies, cover art, and 3D mockups.
- Sales and Distribution: Generate a sales link which can integrate with Kiwify, Hotmart, Shopify, Gumroad or your own app to allow selling your ebook.
- Monetization Dashboard: Let the user manage the pricing, promotional pricing, launch coupons and upsell/downsell configurations, for each AI-Book, to help the creator to properly monetize their AI-book

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for creativity and sophistication.
- Background color: Very light lavender (#F3E5F5), nearly white but slightly tinted to harmonize with the primary.
- Accent color: Violet (#9575CD), a lighter and brighter hue than the primary for CTAs and highlights.
- Headline font: 'Belleza', sans-serif. Body text: 'Alegreya', serif.
- Code font: 'Source Code Pro' for displaying configuration and integration snippets.
- Modern, minimalist icons, related to content creation and AI.
- Subtle animations for generating previews and feedback, and providing a lively feel to the page without being distracting.
